# Step 3: Design System

## Goal

Create a reusable design system for the platform.

## Visual Style

The UI should feel:

- Modern
- Clean
- Student-friendly
- Calm
- Animated
- Premium
- Easy to read

## Theme Direction

Use:

- Rounded corners
- Soft shadows
- Smooth transitions
- Gradient primary buttons
- Neutral background
- Dark mode support if possible

---

## Color Tokens

Create basic color tokens.

Example:

```txt
Primary: Indigo/Violet gradient
Background: Light neutral or dark neutral
Surface: Card background
Text Primary: Main text
Text Secondary: Muted text
Success: Green
Warning: Amber
Error: Red
Border: Subtle border color
```

---

## Typography

Use clear typography.

Recommended scale:

```txt
H1: Page title
H2: Section title
H3: Card title
Body: Normal text
Small: Helper text
```

Font should be readable and modern.

---

```txt
4px

32px
48px
64px
```
